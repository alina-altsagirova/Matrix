# S21_matrix

Implementation of the `matrix.h` library in C. This project provides a comprehensive set of operations for manipulating matrices, developed with a focus on precision and structural integrity.

* **Language & Standard:** Developed in **C11** .
* **Standards Compliance:** Follows **POSIX.1-2017** and **Google C++ Style Guide**.
* **Architecture:** Implemented as a static library `s21_matrix.a` with the `s21_matrix.h` header.
* **Precision:** Supports fractional parts with verifiable accuracy up to **6 decimal places**.
* **Principles:** Adheres to structured programming;
* **Core:** `create_matrix`, `remove_matrix`
* **Comparison:** `eq_matrix`
* **Arithmetic:** `sum_matrix`, `sub_matrix`, `mult_matrix`, `mult_number`
* **Advanced:** `transpose`, `determinant`, `calc_complements`, `inverse_matrix`

## Testing & Quality Assurance

* **Unit Testing:** Full coverage using the **Check** library.
* **Code Coverage:** Minimum **80%** coverage per function, verified via `gcov`.
* **Clean Code:** No legacy or obsolete language constructions used.
